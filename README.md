# Sales Data Analysis – Python & Power BI

Projeto de análise de dados de vendas com exploração em Python e visualização executiva em Power BI.

## Objetivo
Analisar performance comercial e identificar padrões de receita.

## Etapas do Projeto

### 1️ Análise em Python
- Tratamento de dados com pandas
- Análise de faturamento
- Identificação de produtos mais vendidos
- Análise temporal de vendas

### 2️ Dashboard Executivo
- Receita total
- Crescimento mensal
- Ticket médio
- Performance por categoria

## Tecnologias
- Python (pandas, matplotlib, seaborn)
- Power BI
- DAX

## Resultados
Geração de insights estratégicos para suporte à decisão comercial.
